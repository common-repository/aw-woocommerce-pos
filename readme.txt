Woocommerce POS adalah Plugin khusus Woocommerce yang digunakan untuk mengintegrasikan ongkos kirim dengan total belanja calon konsumen Anda.

== Description ==

Woocommerce POS Shipping mengintegrasikan ongkos pengiriman dengan total belanja konsumen Anda pada saat checkout. Dengan beragam fitur yang ditawarkan, plugin ini sungguh powerful dan menjadikan plugin ini wajib digunakan ke semua toko online yang menggunakan wordpress.

= WooCommerce POS Free Version =

* Ongkos kirim POS Kilat
* Data kota POS Jakarta ke Seluruh Indonesia

= WooCommerce POS Full Version =

* Ongkir kirim POS Kilat & POS Express
* Data kota POS dari basis kota pengiriman yang dipilih ke Seluruh Indonesia
* Checkout berdasarkan kode pos
* Biaya Tambahan ( untuk Packing dll )
* Fitur Kalkulasi Ongkir di Halaman Cart
* Gratis update dan Bantuan Support

= WooCommerce POS Exclusive Version =

* Ongkir kirim POS Kilat & POS Express
* Data kota POS dari basis kota pengiriman yang dipilih ke Seluruh Indonesia
* Checkout berdasarkan kode pos
* Biaya Tambahan ( untuk Packing dll )
* Fitur Kalkulasi Ongkir di Halaman Cart
* Tracking mengirimkan no resi langsung ke email pelanggan dari halaman order woocommerce. ( Lacak Pengiriman )
* Free Shipping. Pengaturan pengiriman gratis dengan pos shipping berdasarkan Minimum Biaya, Tanggal, Kota
* Gratis update dan Bantuan Support

== Installation ==

Dibawah ini adalah 3 cara mudah untuk melakukan instalasi pada plugin WooCommerce Kode Pembayaran

1. Untuk menginstal plugin caranya cukup mudah. Apabila Anda ingin mengupload langsung menggunakan file zip. Melalui wordpress pada menu Plugins > Add New > Upload Plugin
2. Aktifkan Plugin
3. Halaman pengaturan plugin di WooCommerce > Settings > POS Shipping
4. Anda sudah bisa menggunakan plugin ini, serta dapat mengkonfigurasi plugin sesuai dengan keinginan Anda.
5. Have fun!
